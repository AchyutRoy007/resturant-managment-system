# Restaurant Management System

## Overview

A simple Python-based restaurant management system that allows users to
view menus, order items, add items, and provide ratings.

## Features

-   Display menu with ratings\
-   Order items & generate bill\
-   Add new menu items\
-   Rate items\
-   JSON-based data storage

## Technologies Used

-   Python\
-   JSON

## Installation & Running

    python main.py

## Testing Instructions

-   Enter valid input options\
-   Test adding items\
-   Test ordering multiple items\
-   Test rating system
